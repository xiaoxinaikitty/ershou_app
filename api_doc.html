商品模块

搜索商品

接口描述：根据关键词搜索商品，支持同时匹配商品标题和商品描述

请求路径：/product/search

请求方法：GET

请求参数说明：

参数名 必选 类型 说明
keyword 否 string 搜索关键词，用于匹配商品标题和描述
pageNum 否 int 当前页码，从1开始，默认1
pageSize 否 int 每页数量，默认10，最大50
categoryId 否 int 商品分类ID筛选
minPrice 否 double 价格下限筛选
maxPrice 否 double 价格上限筛选
sortField 否 string 排序字段(price-价格, time-时间, view-浏览量)，默认time
sortOrder 否 string 排序方式(asc-升序, desc-降序)，默认desc
status 否 int 商品状态筛选(0下架 1在售 2已售)，默认1
minConditionLevel 否 int 最低成色等级筛选(1-10)
location 否 string 地区筛选

请求数据模型：

// ProductSearchDao.java
public class ProductSearchDao {

     * 当前页码，从1开始
     */
    private Integer pageNum = 1;
    
    /**
     * 每页数量，默认10
     */
    private Integer pageSize = 10;
    
    /**
     * 搜索关键词，用于匹配商品标题和描述
     */
    private String keyword;
    
    /**
     * 商品分类ID
     */
    private Integer categoryId;
    
    /**
     * 价格下限
     */
    private Double minPrice;
    
    /**
     * 价格上限
     */
    private Double maxPrice;
    
    /**
     * 排序字段 (price-价格, time-时间, view-浏览量)
     */
    private String sortField = "time";
    
    /**
     * 排序方式 (asc-升序, desc-降序)
     */
    private String sortOrder = "desc";
    
    /**
     * 商品状态筛选 (0下架 1在售 2已售)
     */
    private Integer status = 1;
    
    /**
     * 最低成色等级(1-10)
     */
    private Integer minConditionLevel;
    
    /**
     * 地区筛选
     */
    private String location;
    
    /**
     * 卖家ID
     */
    private Long sellerId;
}

请求示例：

GET /product/list?pageNum=1&pageSize=10&categoryId=2&minPrice=1000&maxPrice=5000&sortField=price&sortOrder=asc

响应参数说明：

参数名 类型 说明
code int 状态码，0表示成功
message string 提示信息
data object 分页结果数据
data.pageNum int 当前页码
data.pageSize int 每页大小
data.total long 总记录数
data.pages int 总页数
data.list array 当前页商品列表
data.hasPrevious boolean 是否有上一页
data.hasNext boolean 是否有下一页

获取商品总数

接口描述：获取系统中符合指定状态的商品总数

请求路径：/product/count

请求方法：GET

请求参数说明：

参数名 必选 类型 说明
status 否 integer 商品状态(0下架 1在售 2已售)，不传表示获取所有状态的商品数量

请求示例：

GET /product/count?status=1

响应参数说明：

参数名 类型 说明
code int 状态码，0表示成功
message string 提示信息
data long 商品总数

响应示例：

{
    "code": 0,
    "data": 123,
    "message": "success"
}

错误码说明：

错误码 说明
0 成功
40000 请求参数错误
50000 系统内部错误

备注：

  1. 此接口不需要用户登录即可访问
  2. status参数不传时，返回所有状态的商品总数
  3. status=1时，返回在售商品总数
  4. status=2时，返回已售商品总数
  5. status=0时，返回下架商品总数

商品支付模块

创建钱包账户

接口描述：为新用户创建钱包账户，初始化余额和冻结余额为0

请求路径：/api/wallet/create

请求方法：POST

请求类型：application/json

请求头：

Authorization: Bearer eyJhbGciOiJIUzI1NiJ9...

请求数据示例：

{
    "userId": 1
}

请求数据模型：

// CreateWalletDTO.java
public class CreateWalletDTO {
    
    /**
     * 用户ID
     */
    @NotNull(message = "用户ID不能为空")
    private Long userId;
}

响应参数说明：

参数名 类型 说明
code int 状态码，0表示成功
message string 提示信息
data long 钱包账户ID（account_id）

响应数据模型：

// BaseResponse.java
public class BaseResponse<T> {
    private int code;
    private T data;
    private String message;
}

响应示例：

{
    "code": 0,
    "data": 123,
    "message": "success"
}

错误码说明：

错误码 说明
0 成功
40000 请求参数错误
40100 未授权
50001 操作失败（如创建钱包失败、用户已有钱包账户等）

备注：

  1. 此接口用于为新用户初始化钱包账户
  2. 同一用户ID只能创建一个钱包账户，重复创建会返回错误
  3. 钱包账户创建后，初始余额和冻结余额都为0
  4. 需要用户登录后访问此接口

查询钱包账户信息

接口描述：根据用户ID查询其钱包账户的详细信息，包括余额、冻结余额等

请求路径：/api/wallet/info

请求方法：GET

请求头：

Authorization: Bearer eyJhbGciOiJIUzI1NiJ9...

请求参数说明：

参数名 必选 类型 说明
userId 是 long 用户ID

请求示例：

GET /api/wallet/info?userId=1

响应参数说明：

参数名 类型 说明
code int 状态码，0表示成功
message string 提示信息
data object 钱包账户信息对象
data.accountId long 钱包账户ID
data.userId long 用户ID
data.balance decimal 账户余额
data.frozenBalance decimal 冻结余额
data.lastUpdateTime datetime 最后更新时间

响应数据模型：

// WalletAccount.java
public class WalletAccount {
    
    /**
     * 钱包账户ID（唯一主键）
     */
    private Long accountId;

    /**
     * 所属用户ID，关联user表的user_id
     */
    private Long userId;

    /**
     * 钱包余额
     */
    private BigDecimal balance;

    /**
     * 冻结余额，如在交易处理中被冻结的金额
     */
    private BigDecimal frozenBalance;

    /**
     * 最后更新时间
     */
    private LocalDateTime lastUpdateTime;
}

响应示例：

{
    "code": 0,
    "data": {
        "accountId": 123,
        "userId": 1,
        "balance": 1500.00,
        "frozenBalance": 200.00,
        "lastUpdateTime": "2023-06-15T10:30:00"
    },
    "message": "success"
}

错误码说明：

错误码 说明
0 成功
40000 请求参数错误
40100 未授权
40400 钱包账户不存在
50000 系统内部错误

备注：

  1. 此接口用于查询用户钱包账户详细信息
  2. 需要用户登录后访问此接口
  3. 如果用户没有创建钱包账户，将返回钱包账户不存在的错误
