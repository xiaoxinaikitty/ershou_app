二手交易平台API文档

用户模块

用户注册

请求路径：/user/register

请求方法：POST

请求数据示例：

{
    "username": "张三",
    "phone": "13800138000",
    "password": "password123",
    "email": "zhangsan@example.com"
}

响应示例：

{
    "code": 0,
    "data": "注册成功",
    "message": "success"
}
